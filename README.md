## Python Syntax 
Python Indentation
```언어
if 5 > 2:
  print("Five is greater than two!")
```
공백을 넣지 않으면 오류가 발생한다. 일반적으로 4개의 공백이 사용되지만 적어도 하는 있어야 한다.
  예
```언어
  if 5 > 2:
 print("Five is greater than two!") 
if 5 > 2:
        print("Five is greater than two!") 
```

##Python Comments
주석은 #으로 시작하며 그 뒤는 파이썬에서 무시된다.
print("Hello, World!") # 주석입니다
위처럼 줄 끝에 배치할 수 있다
Multiline Comments
삼중 따옴표(```)를 사용하여 그 안에 주석을 넣을 수 있다.
"""
안녕하세요
감사합니다
잘 부탁드립니다
"""
print("Hello, World!")
